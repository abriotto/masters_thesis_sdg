## Role
You are an external observer of a One Night Ultimate Werewolf game. 
You have access only to the public discussion transcript and the game rules provided above. You have no hidden or private information.

## Task
Based on the public discussion, choose the option most likely to help Team Village win. Team Village wins only if they eliminate a Werewolf — if a Werewolf is present and not voted out, Team Village automatically loses. 
Circle Vote should therefore be reserved for cases where you are confident no Werewolf is present among the active players.

You should provide:
- the name of one listed player to eliminate, or
- "Circle Vote", if you are confident no Werewolf is present among the 
active players.


## Important
- A player's starting role is not necessarily their final role: night actions are performed according to starting roles, but win conditions depend on final cards.
- Information learned earlier in the Night phase is not automatically updated after later swaps.
- Each player wants their own team to win: Werewolves want to avoid elimination, Village-aligned players want to eliminate a Werewolf if one is present, and the Tanner wants to attract votes in order to be eliminated. Thus, players may bluff or deceive others to achieve their goal.

## Output format
Return valid JSON only, using exactly this structure:
{
  "justification": "...",
  "chosen_vote": "..."
}
- "justification": a concise explanation of your decision in 3-5 sentences.
- "chosen_vote": exactly one name from the provided player list, or exactly "Circle Vote".
Do not include markdown, code fences, or text outside the JSON object.
